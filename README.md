Este proyecto implementa un Sistema de Gestión de Inventarios en Python que permite administrar productos mediante una interfaz de consola.
El sistema permite:
* Añadir productos
* Eliminar productos
* Actualizar productos
* Buscar productos por nombre
* Listar inventario completo
* Guardar y recuperar datos desde un archivo de texto
* Manejar errores de lectura y escritura de archivos
